# A post about Bitcoins

Everybody is writing about it, so am I even though I don't own anything from this rabid sea.
I guess this post should begin with a bit of history about the coin which isn't a coin. The problem though is: there is no real history. The concept of this currency (whether or not this term is correct) was published in 2007 under an nom de plume, an alias. Any traces of the author ever since lead into the dark. To lower any unease instantly arising from this it must be said that cryptographers all around the world looked into the source. The source thereby is publicly available, can be forked, can be changed. A good thing, how can one be uncomfortable with something being completely transparent and open. Fair enough, but what if experts look at the code and hostly admit that this piece of art can't be written by one guy. Too much, too much knowledge involved to be built by one soul. So it must be a team of dudes being organized… an organization. That's almost a synonym to conspiracy in most circles these days.
Apart from all anxiety, this project is a great experiment which deserves some discussion.
